Online Web Development Competition

Posted by Coding Club

The Coding Club IITG is proud to present the online web development competition - BACK TO THE PAST!
It is an opportunity for all first year students of the college to develop a game from the given list, to enhance their web development skills, and to earn amazing prizes such as:
1) First Prize: Gaming Keyboard
2) Second Prize: Gaming Mouse
3) Third Prize: Speakers
General Rules:
The game's interface should be built on HTML,CSS,JS,etc. web design languages/frameworks.
The project has to be on an open source repository.
There will be bonus points for a well maintained repository (issues, README.md,etc.).
The game should one from the following given list :
1.) Snake
2.) Flappy Birds
3.) Tetris
4.) Some endless Run (like Stick Run, etc.)
If you want some game to be added in the list please contact the organisers.
The master branch of the repository as of 21st June'17 4:59PM will be the final submission. i.e.You have 10 days time to finish your project.
Judging Criteria:
Points shall be awarded on the basis of the following factors:
Novelty of the idea.
Proper working of the game.
Smoothness of animation.
UI / UX.
Game should be accessible on Chrome/Firefox (If not a standalone web based app).
